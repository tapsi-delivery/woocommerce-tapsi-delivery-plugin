== Description ==

Seamlessly integrate your WooCommerce website with Tapsi Pack Delivery!
Tapsi Pack is a powerful AI-driven delivery service designed to enhance the shipping experience for WooCommerce shops. This plugin enables shoppers to add their locations and available time slots, allowing customers to choose the origin location, date, and time for package delivery during checkout.

== Features ==

- Seamless integration with WooCommerce.
- Shoppers can add multiple locations and specify available time slots.
- Customers can choose the origin location during checkout.
- Select a preferred date and time slot for package delivery.

== Installation ==

1. Upload the `tapsi-pack` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings in the WooCommerce settings page.

== Configuration ==

1. Navigate to the WooCommerce settings page.
2. Look for the "Tapsi Pack" tab.
3. Login to your Tapsi Pack account.
4. Configure your delivery settings, including your locations.

== Usage ==

1. Shoppers can add their locations and available time slots in their account settings.
2. During checkout, customers can choose the origin location from the available shopper locations.
3. Select a preferred date and time slot for package delivery.
4. Complete the checkout process as usual.

== Frequently Asked Questions ==

= Can I use Tapsi Pack with other shipping methods? =

Yes, Tapsi Pack can be used alongside other shipping methods in WooCommerce.

= How do shoppers add their locations and time slots? =

Shoppers can log in to their accounts and navigate to the account settings to add and manage their locations and time slots.
